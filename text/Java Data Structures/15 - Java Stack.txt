What is a Stack?
- A Stack is a linear data structure that follows the LIFO (Last In, First Out) principle.
- The last element inserted is the first one removed.
- Real-life examples:
- A pile of plates
- Browser back button history
- Undo operations in editors

Basic Stack Operations

Operation	        Description
push(x)	            Insert element x on top
pop()	            Remove and return top element
peek() / top()	    Return top element without removing
isEmpty()	        Check if stack is empty
size()	            Return number of elements


Stack Implementation Using Javaâ€™s Built-in Class
- Java provides a stack in java.util.Stack;
- Example 

import java.util.Stack;

public class Main() {
    public static void (String args[]) {
        Stack<Integer> stack = new Stack<Integer>();

        stack.push(10);
        stack.push(20);
        stack.push(30);

        System.out.println("Top: " + stack.peek());  // 30
        System.out.println("Popped: " + stack.pop()); // 30
        System.out.println("Is Empty: " + stack.isEmpty()); // false
        System.out.println("Size: " + stack.size()); // 2
    }
}

Stack Implementation Using Array (Manual Implementation)

class MyStack {
    int[] arr;
    int top;
    int capacity;

    MyStack(int size) {
        arr = new int[size];
        top = -1;
        capacity = size;
    }

    public void push(int x) {
        if (top == capacity - 1) {
            System.out.println("Stack Overflow!");
            return;
        }
        arr[++top] = x;
    }

    public void pop() {
        if (top == - 1) {
            System.out.println("Stack Underflow!");
            return;
        }

        return arr.remove(top--);
    }

    public void peek() {
        if (top == - 1) {
            System.out.println("Stack Underflow!");
            return;
        }

        return arr[top];
    }

    public boolean isEmpty() {
        return top == -1;
    }

    public boolean size() {
        return top + 1;
    }
}

public class StackArrayDemo {
    public static void main(String[] args) {
        MyStack s = new MyStack(5);

        s.push(10);
        s.push(20);
        s.push(30);

        System.out.println("Top: " + s.peek());
        System.out.println("Popped: " + s.pop());
        System.out.println("Size: " + s.size());
    }
}

Stack Implementation Using Linked List

class Node {
    int data;
    Node next;

    Node(int data) {
        this.data = data;
    }
}

class StackLL {
    Node top = null;

    public void push(int x) {
        Node newNode = new Node(x);
        newNode.next = top;
        top = newNode;
    }

    public int pop() {
        if (top == null) {
            System.out.println("Stack Underflow!");
            return -1;
        }
        int val = top.data;
        top = top.next;
        return val;
    }

    public int peek() {
        if (top == null) return -1;
        return top.data;
    }

    public boolean isEmpty() {
        return top == null;
    }
}

public class StackLLDemo {
    public static void main(String[] args) {
        StackLL s = new StackLL();

        s.push(5);
        s.push(15);
        s.push(25);

        System.out.println(s.pop());
        System.out.println(s.peek());
    }
}
